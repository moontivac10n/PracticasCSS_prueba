Aprendiendo a mejorar el aspecto del HTML con CSS

En HTML, puedes crear listas con viñetas (también conocidas como listas con viñetas o listas desordenadas) utilizando la etiqueta "ul" (unordered list) y los elementos "li" (list item) para cada elemento de la lista. Compatibilidad con Navegadores: Menciona que, aunque CSS es ampliamente compatible, algunos navegadores pueden interpretar ciertas propiedades de manera diferente. Anima a los estudiantes a realizar pruebas en diferentes navegadores.

Para utilizar viñetas en HTML debes usar la etiqueta ul y dentro de la etiqueta ul agregar las etiquetas

  • id="elemento1"
  • id="elemento2"
  • id="elemento3"
  • id="elemento4"
